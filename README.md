# 随机网名生成器

一个简洁美观的随机网名生成网站工具，支持多种风格的网名生成和一键复制功能。

## ✨ 功能特点

- 🎨 **多种风格选择**：古风、现代、可爱、酷炫、文艺等多种风格
- 🎲 **随机生成**：每次点击生成10个不同的网名
- 📋 **一键复制**：点击网名即可复制到剪贴板
- 📱 **响应式设计**：适配各种屏幕尺寸的设备
- 🌈 **美观界面**：渐变色彩和流畅动画效果

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计（渐变、动画、响应式）
- **JavaScript**：功能实现
- **纯前端**：无需后端服务器，直接打开即可使用

## 🚀 快速开始

### 方法一：直接访问

1. 克隆或下载本项目
2. 双击打开 `index.html` 文件即可使用

### 方法二：本地服务器（推荐）

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve

# 然后在浏览器中访问 http://localhost:8000
```

## 📖 使用说明

1. 在"选择风格"下拉菜单中选择你喜欢的网名风格
2. 点击"✨ 生成网名"按钮
3. 浏览生成的网名列表
4. 点击任意网名即可复制到剪贴板（会显示"已复制！"提示）

## 🎯 自定义扩展

### 添加更多网名

你可以在 `script.js` 文件中的 `nameDatabase` 对象中添加更多网名：

```javascript
const nameDatabase = {
    ancient: [
        '现有网名1',
        '现有网名2',
        '你的新网名'  // 添加新网名
    ],
    // 其他风格同理
};
```

### 修改生成数量

在 `script.js` 文件中修改 `generateNames` 函数的默认参数：

```javascript
function generateNames(style = 'all', count = 10) {  // 将10改为你想要的数量
    // ...
}
```
